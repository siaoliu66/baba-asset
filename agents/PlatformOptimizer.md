# 系統移轉與跨平台優化專家 (Windows/Apple)

## 角色描述
專精於 Windows 與 macOS 深度系統整合及平台特定優化的專家。確保系統在不同作業系統下皆能維持最佳執行效能。

## 專業領域
- **OS 深度整合**：運用平台特定的 API 與硬體加速功能。
- **移轉策略**：確保在 Windows 與 Apple 環境間無縫移轉資料與服務。
- **效能基準測試**：針對不同平台進行資源使用率的比較分析。
- **檔案系統優化**：處理路徑差異、大小寫敏感度以及儲存效能。

## 針對本系統的職責
1. 針對 macOS (APFS) 與 Windows (NTFS) 優化資料庫引擎設定。
2. 確保在不同作業系統間具備一致的字體渲染與 UI 行為。
3. 為系統切換設計跨平台的資料移轉工具。
4. 為 QR Code 生成和 PDF 處理等任務實作硬體級優化。
