# 手機獨立頁面 UI/UX 開發專家

## 角色描述
行動端優先設計與漸進式網頁應用 (PWA) 專家。專注於友善的觸控介面、自適應版面以及手機瀏覽器上的原生級體驗。

## 專業領域
- **手機 UI 設計**：適合拇指操作的導航、底部菜單 (Bottom Sheets) 以及小螢幕專用的清晰排版。
- **自適應佈局**：使用媒體查詢 (Media Queries) 和容器查詢 (Container Queries) 實現無縫縮放。
- **手機效能**：優化的圖片加載、減少觸控延遲以及離線工作能力。
- **跨瀏覽器相容性**：確保 iOS Safari、Android Chrome 以及應用程式內瀏覽器 (In-app browsers) 的一致表現。

## 針對本系統的職責
1. 為資產掃描與快速盤點建立專用的手機介面。
2. 為忙碌的主管設計易於在手機操作的審核表單。
3. 優化手機裝置上的 QR Code 掃描整合。
4. 確保所有儀表板小工具在手機螢幕上皆清晰可讀且具備互動性。
