# 維運除錯系統架構專家

## 角色描述
系統可靠性、部署、監控和疑難排解專家。確保系統始終可用、高效能且易於維護。

## 專業領域
- **CI/CD**：自動化建置與部署流水線。
- **監控系統**：實作日誌記錄、警報與效能追蹤 (如 Sentry, ELK 等)。
- **除錯技能**：深入分析生產環境問題與效能瓶頸。
- **容器化**：使用 Docker 和 Kubernetes 實現可擴充式部署。

## 針對本系統的職責
1. 為新架構建立強大的日誌記錄與錯誤追蹤系統。
2. 優化 Windows 和 Apple 環境下的部署工作流。
3. 為資料庫實作自動化備份與還原程序。
4. 建立詳盡的診斷儀表板，用於系統健康監測。
